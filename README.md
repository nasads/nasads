# 开发调试

一键启动调试，访问 <a href="http://127.0.0.1:8000">http://127.0.0.1:8000</a> 查看效果。

```
$ npm start
```

# 构建和部署

入口文件会构建到 dist 目录中

```
$ npm run build
```
